Menu pada Microsoft Office Word 365 (Android) dan cara menggunakannya

Minggu, 05 Feb 2023

Zaman sekarang, orang-orang sudah banyak yang menggunakan aplikasi desktop yang akan kita bahas sekarang, yaitu Microsoft Office. Mulai dari Microsoft Word, Excel, Powerpoint, dan aplikasi produk Microsoft lain sudah banyak digunakan orang-orang untuk keperluan pekerjaannya.

Aplikasi ini umumnya dapat diakses dan digunakan pada perangkat berupa komputer dan laptop. Selain lewat perangkat tadi, Microsoft Office juga tersedia di smartphone berbasis Android lho! Cocok sekali untuk kamu yang ingin belajar Microsoft Office, tetapi belum memiliki komputer atau laptop.

Oke, pada pembahasan kali ini, kita akan membahas tentang Cara Menggunakan Microsoft Office Word 365, yaitu update terbaru dari aplikasi Microsoft versi sebelumnya. Let's go!

A. Cara masuk ke Microsoft Office Word 365

1. Masuk ke aplikasi Microsoft Office 365 yang sudah diunduh. Rata-rata orang sudah tahu lah ya cara mengunduhnya.

2. Setelah masuk ke aplikasi, kurang lebih tampilannya seperti ini.

3. Klik " Create" untuk membuat file baru

4. Setelah mengklik tombol "Create", tampilannya akan seperti ini

5. Klik logo Microsoft Word

6. Ada 4 menu opsi untuk membuat dokumen Microsoft Word

Keterangan :

  • Scan Text, yaitu opsi untuk membuat file dokumen word dengan memindai (Scan) foto/screenshot. Nantinya teks pada foto yang dipindai akan dibaca oleh kamera dan diubah ke dalam bentuk dokumen word.
  • Dictate, yaitu opsi untuk membuat file dokumen word dengan cara pengguna mengeluarkan suara untuk direkam oleh sistem perekam. Hasil audio tersebut akan diubah menjadi bentuk file berupa teks yang akan ditampilkan pada dokumen word.
  • Blank document, opsi yang paling umum digunakan. Opsi ini hanya berisikan lembaran file kosong yang dapat kita isi dengan cara mengetik atau bahkan menyisipkan file berupa gambar dan video.
  • Create from template, opsi ini cukup spesial karena jika kita memilih opsi ini, terlebih dahulu kita harus memilih template, yaitu desain tampilan dokumen word yang menarik, sehingga dokumen word kita tidak monoton warna hitam putih.

7. Untuk tutorial kali ini, kita akan memilih opsi 'Blank document'

8. Beginilah tampilan awal Microsoft Word bagian Tab Home

9. Selesai, kalian sudah bisa mengakses Microsoft Word 365 di Android!
